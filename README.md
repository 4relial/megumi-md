# MEGUMI-MD

![](./assets/smile.gif)

## INSTALLATION

##### CLONE REPOSITORIES
```Alpine Abuild
git clone https://github.com/4relial/megumi-md
cd megumi-md
```

#####  INSTALL THE DEPENDENCIES 
```Alpine Abuild
npm i
```

#####  START BOT
```Alpine Abuild
node index.js1
```
